# Xml (XHTML) Library


This module contains:
* an [XML Cli](cli) to perform XML operations at the command line
* an [XML library](library-xml) that implements XML functions.
* an [HTML library](library-html/README.md) that implements HTML functions (Jsoup, CssParser)


## Installation


For Linux/Windows WSL and macOS, with [brew](https://brew.sh/)

```bash
# install
brew install gerardnico/tap/xmli
# uninstall
brew uninstall xmli
```
